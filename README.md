# Flutter Projects by Rajnish Shrestha

Welcome to my Flutter projects repository! This repo contains **5 complete Flutter applications** and an **ongoing project**, showcasing my skills in mobile app development, state management, local and cloud storage, REST APIs, secure real-time communication, and payment integration.

---

## Projects

### 1. **Notes App**
- **Description:** A note-taking app with local data storage using **SQFlite**.
- **Features:**
  - Add, edit, and delete notes
  - Persistent local storage
  - Clean and responsive UI

---

### 2. **Task Management App**
- **Description:** A cloud-synced task management app using **Firebase**.
- **Features:**
  - Add, update, delete tasks
  - Real-time cloud sync
  - User authentication
  - Organized task categories

---

### 3. **E-commerce App**
- **Description:** A fully-featured e-commerce app with **BLoC architecture**.
- **Features:**
  - Browse products and categories
  - Add to cart & manage favourites
  - Payment integration with **Khalti**
  - Clean UI with smooth navigation
  - State management using **BLoC**

---

### 4. **Weather App**
- **Description:** A weather forecast app using **REST APIs** and device location.
- **Features:**
  - Get current weather using **OpenWeatherMap API**
  - Fetch weather based on device location (**Geolocator**)
  - User-friendly UI with dynamic icons and temperature display

---

### 5. **Chatting App**
- **Description:** A secure real-time messaging app.
- **Features:**
  - End-to-end encryption using **AES**
  - Real-time messaging with **Firebase**
  - Group chats
  - Smooth and responsive interface

---

### 6. **Servup (Ongoing Project)**
- **Description:** A local services platform connecting users with service providers.
- **Features (in progress):**
  - User authentication and profiles 
  - Service booking system
  - Clean, responsive, and structured UI
  - Smooth workflow for real-world app functionality
